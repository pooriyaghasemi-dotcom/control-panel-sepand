# کنترل پنل کارخانه سپند

سیستم اتوماسیون و امنیتی کارخانه سپند با سه برد ESP32

## ویژگی‌ها

### سیستم‌های اصلی:
- 🔒 **سیستم امنیتی:** سنسورهای لیزری، سنسورهای درب، آژیر
- 💡 **کنترل روشنایی و کولر:** رله‌های ۱۶ کانال و ۴ کانال
- 📊 **نمایش سنسورها:** دما، رطوبت
- 🌐 **Web Panel:** کنترل و نمایش از طریق مرورگر

### برد‌ها:

1. **برد سالن تولید (Main Server)**
   - Web Server + Web Panel
   - ۱۶ کانال رله
   - سنسور DHT22
   - میکروسویچ درب
   - سایرن/آژیر (رله ۱)

2. **برد دفتر (Client)**
   - ۴ کانال رله
   - سنسور DHT22
   - میکروسویچ درب

3. **برد حیاط (Client)**
   - ۴ کانال رله
   - سنسور DHT22
   - میکروسویچ درب ورودی ۱ و ۲
   - ۳ سنسور لیزری

## تنظیمات WiFi

- هر برد یک WiFi AP دارد برای Setup اولیه
- تمام برد‌ها به WiFi کارخانه متصل می‌شوند
- Static IP برای هر برد

## Web Panel

- Login با Username و Password
- دو مود: کار فعال و سیستم امنیتی فعال
- کنترل رله‌ها
- نمایش وضعیت سنسورها
- نمایش سابقه رویدادها
